# Pink Glam Makeup Planner

A cute, pink-themed web app to help you organize and plan your makeup looks. Save ideas, track products, and mark your favorite glam looks — all in one place.

---

## Features

- **Create & Manage Looks** — Add makeup looks with a name, occasion, notes/steps, and color vibe
- **Product Tracking** — Tag products used in each look; view all tracked products in the Products tab
- **Favorites** — Heart any look to mark it as a favorite and filter by favorites
- **Search & Filter** — Search by look name, notes, or products; filter by occasion
- **Stats Dashboard** — See your total looks, favorites, and unique products at a glance
- **Local Storage** — All data is saved in your browser automatically, no account needed
- **Responsive Design** — Works on desktop and mobile

---

## Tech Stack

- HTML5
- CSS3 (custom properties, flexbox, grid, animations)
- Vanilla JavaScript
- LocalStorage API

No frameworks, no dependencies, no build step — just open and go.
